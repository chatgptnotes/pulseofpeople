# IMPORTANT NOTES
# =============================================================================

1. VITE_APP_URL: Update this to your actual Render URL after deployment
   - Initial: https://pulseofpeople.onrender.com
   - Or use custom domain: https://yourdomain.com

2. Security:
   - Change DEBUG=False for production backend
   - Generate a new SECRET_KEY for production (use: python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())")
   - Use DB_SSLMODE=require (not prefer) for production

3. After Adding Variables:
   - Click "Save Changes"
   - Trigger "Manual Deploy" > "Clear build cache & deploy"
   - Wait for build to complete (2-3 minutes)

4. Verification:
   - Visit your Render URL
   - Check browser console for errors
   - Verify Supabase connection works
   - Test dashboard data loading

5. Custom Domain (Optional):
   - Add custom domain in Render settings
   - Update VITE_APP_URL to match your domain
   - Redeploy after changing
